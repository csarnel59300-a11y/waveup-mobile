Politique de Confidentialité - WaveUp

Dernière mise à jour : Décembre 2024

1. Données collectées

WaveUp collecte les données suivantes via TikTok OAuth :

  • Nom d'utilisateur TikTok
  • Photo de profil
  • Statistiques publiques du compte

2. Utilisation des données

Vos données sont utilisées uniquement pour :

  • Afficher vos statistiques TikTok
  • Générer des recommandations personnalisées
  • Améliorer votre expérience utilisateur

3. Partage des données

Nous ne vendons ni ne partageons vos données avec des tiers.

4. Suppression des données

Pour demander la suppression de vos données, contactez-nous à : waveupai@gmail.com

Les données sont automatiquement supprimées après 90 jours d'inactivité.
